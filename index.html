  1. Drie eigenschappen van een leesbare tekst:
    • Tekst is minimaal 16px groot.
    • Regelafstand bij langere teksten is 1.4 (140%).
    • Optimale regellengte: 10–12 woorden.
    • https://www.w3schools.com/html/html5_syntax.asp
  2. Stappen voor mobile-first werken:
    • Schrijf eerst CSS voor het kleinste scherm (basislayout).
    • Voeg breakpoints toe met @media (min-width: …) en pas stijlen aan voor grotere schermen.
    • Pas layout en content per breakpoint aan (meer kolommen, grotere fonts, extra elementen).
    • Herkenbaar in code: basisregels zijn mobiel; media queries staan erna met min-width.

Sprint 7 – Connect Your Tribe
02-02-2026
  1. Hoe link je bestanden vanuit je public map?
    Door een / voor het pad te zetten, bijvoorbeeld /style/style.css.
  2. Wanneer moet je je server herstarten?
    Als je iets verandert in server.js.
  3. Wat moet je doen om je server te herstarten?
    Stop de server met ctrl + c en start hem opnieuw met npm start.
  4. Heb je syntax highlighting voor liquid code aan staan?
    Ja, met de Shopify Liquid extensie.
  5. Hoe open je de terminal met je keyboard?
    Met ctrl + `.
  6. Wat staat er in de node_modules map?
    Alle packages die je project gebruikt.
  7. Welke liquid filters heb je al gebruikt?
    Ik heb {{ person.name }} gebruikt om data op te halen.
04-02-2026
  1. Hoe zou je Liquid beschrijven?
    Het is een taal die dynamische data kan ophalen uit een database.
  2. Wat kun je met Liquid filters doen?
    Data ophalen uit de database.
  3. Kun je Liquid code in de browser zien als je de HTML broncode bekijkt?
    Nee, de code verandert in een string met tekst.
  4. Waar wordt Liquid code uitgevoerd?
    Op een server.
  5. Waar wordt HTML uitgevoerd?
    In de browser.
  6. Wat is een route?
    Stukjes achter de URL waar je site op draait.
  7. Wat hebben routes met bestandsnamen te maken?
    Routes kun je koppelen aan bestandsnamen met een callback functie. Elke keer dat een bezoeker een bepaalde route bezoekt, wordt de callback functie uitgevoerd. In die functie wordt een bestandsnaam gerenderd.
  8. Kun je in je browser zien welke bestandsnamen je voor views hebt gebruikt, kon dat vorig semester wel?
    Ja, bij de inspector mode onder het kopje 'Network'. Dat kon vorig semester niet, want toen hadden we onze content niet op een server draaien.
  9. Wat doet JSON.parse()?
    Zorgt dat een string met tekst gelezen kan worden in JS als JSON.
11-02-2026
  1. Wat is een query parameter?
    Extra informatie in de URL die naar de server wordt gestuurd.
  2. Op wat voor soort data kun je for loops gebruiken?
    Op arrays, dus lijsten met data.
  3. Welke Liquid tags ken je inmiddels?
    {% if %}, {% endif %}, {% for %}, {% endfor %}, {% else %}, {% elsif %}.
  4. Welke Liquid filters heb je al gebruikt?
    Ik heb een date filter gebruikt om een datum anders weer te geven.
  5. Wat versta jij onder URL design, en nette URLs?
    Korte en duidelijke URLs, zoals example.com/running-shoes.
  6. Wat vind je lelijke URLs?
    Lange en onduidelijke URLs, zoals example.com/?id=123&sort=desc.
